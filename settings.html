Group ID 与 API 密钥
均可在 MiniMax 开放平台「账户管理」中查看，仅保存在本机浏览器内，不会上传至任何服务器。

生成的新音色为临时音色，需在 168 小时内于任意语音合成接口调用一次才能永久保留，超时未使用会被自动清除。
